Basic Information

Clicking on any button in the top bar will change the editor interface.
The left panel contains an interface to customize and add game changes.
The middle panel contains a summary of your changes, and you can click on the text to remove your change.
The right panel contains the code that you need to add to either Game.ini, GameUserSettings.ini, or both.
As of right now, the only implemented tool is the spawn rate modifier. It cannot create configs for bosses/non-natural-spawning mobs. It does support mob addition and replacement.
